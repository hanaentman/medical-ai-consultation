[이미지 첨부 운영 기준]
이 문서는 AI 상담 답변에 이미지를 첨부할 때 참고할 운영 기준입니다.
답변 내용은 병원 문서를 우선 근거로 하며, 이미지는 사용자의 질문 해결에 직접 도움이 되는 경우에만 첨부합니다.

[공통 원칙]
- 이미지는 사용자가 요청한 정보와 직접 관련이 있을 때만 첨부한다.
- 사용자가 내부 시설 위치, 진료 흐름, 검사실 위치처럼 짧은 안내만 필요한 질문을 한 경우에는 불필요한 이미지를 첨부하지 않는다.
- 같은 답변에 여러 이미지를 붙여야 할 때는 사용자의 질문 의도와 가장 직접 관련 있는 이미지를 우선한다.
- 이미지가 첨부되더라도 답변 본문에는 핵심 내용을 텍스트로 먼저 설명한다.

[CI.png]
첨부 조건:
- 메인 페이지 상단 로고로 사용한다.

첨부 제외:
- 채팅 답변에는 일반적으로 첨부하지 않는다.

[병원 약도.png]
첨부 조건:
- 병원 전체 위치를 묻는 질문
- 병원 오시는 길을 묻는 질문
- 병원 주소, 지도, 찾아가는 길, 대중교통, 주차장 진입 안내를 묻는 질문
- 병원 외부에서 병원까지 어떻게 오는지 묻는 질문

첨부 제외:
- 병원 내부 시설 위치를 묻는 질문
- 외부CT실, CT실, 검사실, 화장실, 원무과, 접수, 수납, 약제실, 채혈실, 입원실, 병동 위치를 묻는 질문
- 특정 층이나 특정 부서 위치를 묻는 질문

예시:
- "병원 어디에 있어?" -> 첨부
- "병원 주소 알려줘" -> 첨부
- "오시는 길 알려줘" -> 첨부
- "외부CT실 어디야?" -> 첨부하지 않음
- "화장실 어디 있어?" -> 첨부하지 않음

[셔틀버스 시간표.png]
첨부 조건:
- 셔틀버스 운행 시간, 탑승 위치, 배차 간격, 순환버스 관련 질문

첨부 제외:
- 일반 대중교통 안내만 묻는 질문
- 병원 내부 시설 위치를 묻는 질문

[입원전 복용중단 약물 리스트.jpg]
첨부 조건:
- 입원 전 복용 중단 약물 문의
- 수술 전 복용 중단 약물 문의
- 아스피린, 와파린, 항응고제, 혈전약, 피를 묽게 하는 약과 관련된 수술 전 문의

첨부 제외:
- 현재 복용 중인 약을 임의로 중단해도 되는지 묻는 질문
- 처방 변경이나 복용 중단을 확정적으로 요구하는 질문

주의:
- 약 복용 중단 여부는 반드시 의료진 확인이 필요하다고 안내한다.
- AI가 특정 약을 중단하라고 지시하지 않는다.

[진료일정전체.png]
첨부 조건:
- 진료 일정, 의료진 일정, 진료표, 휴진, 토요일 진료, 예약 가능 일정 관련 질문

첨부 제외:
- 단순 진료시간 문의
- 특정 증상에 대해 어느 센터로 가야 하는지 묻는 질문

[의료진 사진]
첨부 조건:
- 답변에서 특정 의료진을 소개하는 경우 해당 의료진 사진을 첨부한다.
- 사용자가 특정 의료진 이름을 직접 질문한 경우 해당 의료진 사진을 첨부한다.

첨부 제외:
- 의료진 전체 목록을 간단히 안내하는 경우에는 모든 사진을 한꺼번에 첨부하지 않는다.
- 문서에 근거 없이 특정 의료진을 임의로 추천하는 경우에는 사진을 첨부하지 않는다.

[내부 시설 위치 질문 처리 기준]
내부 시설 위치 질문은 병원 약도보다 텍스트 안내를 우선한다.

예시 질문:
- 외부CT실 어디야?
- CT실은 어디로 가나요?
- 화장실 어디 있어?
- 원무과 어디야?
- 접수는 어디서 해?
- 수납은 어디서 해?
- 검사실은 몇 층이야?

답변 기준:
- 문서에 위치 정보가 있으면 해당 층과 이동 방법을 짧게 안내한다.
- 문서에 위치 정보가 없으면 원무과 또는 현장 직원에게 확인하라고 안내한다.
- 병원 전체 약도는 첨부하지 않는다.
